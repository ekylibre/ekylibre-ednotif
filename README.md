# Ednotif

Plugin for integrate animal's automatic data.

You need a cipher key to operate this plugin into your config/application.yml file in Ekylibre

rake cipher:aes

'INTEGRATION_CIPHER_BASE64_KEY = "YOUR KEY"'

Code entreprise : 'E170'
